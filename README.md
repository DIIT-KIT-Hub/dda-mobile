## Концепт гри Bug Detective

**Bug Detective** — це 2D-гра-головоломка для Android у нуарному стилі (чорно-біле місто з кольоровими акцентами), у якій гравець виступає у ролі **детектива з відділу програмних розслідувань** і розкриває злочини, спричинені багами у програмному коді.

Гра одночасно:
- **Розважає** — нуарний наратив, NPC, прогрес кейсів, ачівменти.
- **Навчає** — кожен кейс відпрацьовує конкретну техніку налагодження (debugging skill): локалізація багу, класифікація, формулювання гіпотези, виправлення.
- **Адаптується** — модель машинного навчання (DDA) на пристрої аналізує поведінку гравця і автоматично підлаштовує складність наступного кейса.
